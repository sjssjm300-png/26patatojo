/* * [화장실 악취 저감을 위한 흡착제 필터 연구] * 대상: 이지메이커 가스 센서 (MQ-135 소자 기반) * 기능: 농도 구배에 따른 확산 및 흡착 효율 실시간 모니터링 */ const int sensorPin = A0; // 이지메이커 센서 아날로그 핀 float Ro = 10.0; // 깨끗한 공기에서의 초기 저항값 (실험 전 보정 필요) void setup() { Serial.begin(9600); Serial.println("--- 암모니아 흡착 실험 데이터 로깅 시작 ---"); Serial.println("시간(sec), 전압(V), 농도비(Rs/Ro)"); // 센서 예열 알림 (최소 5분 권장) delay(1000); } void loop() { // 1. 데이터 평활화 (노이즈 제거를 위해 10번 읽어 평균 계산) long sum = 0; for(int i=0; i<10; i++) { sum += analogRead(sensorPin); delay(10); } float sensorValue = sum / 10.0; // 2. 전압값 변환 (0~5V) float voltage = sensorValue * (5.0 / 1023.0); // 3. 센서 저항(Rs) 계산 // RL(부하저항)은 모듈 특성에 따라 다르지만 보통 10kOhm 내외 float Rs = ((5.0 * 10.0) / voltage) - 10.0; // 4. 농도비 계산 (Rs/Ro) // 확산에 의해 농도가 변하면 이 비율이 가파르게 변함 float ratio = Rs / Ro; // 5. 시리얼 플로터용 데이터 출력 // 엑셀에 붙여넣기 좋게 콤마(,)로 구분 Serial.print(millis() / 1000); // 경과 시간(초) Serial.print(", "); Serial.print(voltage); // 전압 변화 (확산도 관찰용) Serial.print(", "); Serial.println(ratio); // 흡착 효율 지표 // 6. 측정 주기 설정 // 농도 구배에 따른 확산을 관찰하려면 너무 빠르지도, 느리지도 않은 주기가 중요 // 초기 5분은 10초 단위, 이후는 30초 단위로 데이터 로깅 권장 delay(10000); }
